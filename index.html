Minha Skills

Conheça as tecnologias que domino:

Hard Skills
  • Python: Desenvolvimento de scripts analíticos utilizando bibliotecas como Pandas e NumPy para manipulação de grandes volumes de dados, e Matplotlib, Seaborn ou Plotly para visualizações dinâmicas e geração de gráficos.
  • SQL & Bancos de Dados: Construção de consultas eficientes para extração e filtragem de dados utilizando PostgreSQL, MySQL e SQL Server, aplicando junções (JOINs), agregações (GROUP BY), subqueries e funções de janela.
  • Power BI: Criação de dashboards dinâmicos e relatórios interativos. Desenvolvimento de modelagem de dados utilizando o padrão Star Schema, criação de métricas e colunas calculadas com linguagem DAX, e tratamento de dados no Power Query.
  • Excel: Tratamento e análise rápida de dados utilizando tabelas dinâmicas, funções avançadas de busca e lógica (PROCV/XLOOKUP, SE/IFS) e criação de gráficos integrados para relatórios executivos.

Skills de Fundamento

  • Tratamento de Dados (Data Wrangling): Identificação e resolução de inconsistências, como registros duplicados, tratamento de valores ausentes (nulos) e conversão de tipos de variáveis para garantir a integridade da base.
  • Análise Exploratória de Dados (EDA): Exploração inicial de novas bases de dados por meio de tabelas de frequência, cruzamento de variáveis e detecção de padrões ocultos ou anomalias (outliers) que direcionam os insights.
  • Estatística Descritiva: Aplicação prática de medidas de tendência central (média, mediana, moda) e de dispersão (variância, desvio padrão, quartis) para resumir, compreender e descrever o comportamento dos dados.
